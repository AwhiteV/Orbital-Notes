## 4. 技术背后的匠心：为什么选择它？

作为一名技术博主，我最欣赏的是它在技术实现上的克制与平衡：

1.  **纯粹的 Vanilla JS**：没有引入 React 或 Vue 等重型框架，保证了应用在 Electron 环境下的毫秒级响应。
2.  **自定义渲染组件**：自研了 `<Note>`（高亮提示）和 `<Accordion>`（折叠面板）扩展，让 Markdown 展示更具层次感。
3.  **多引擎导出**：针对 Word 使用 `docx` 库生成标准文档，针对 PDF 调用 `Puppeteer` 进行排版打印，确保各平台排版一致。
4.  **离线存储优先**：基于 JSON 格式的本地持久化，数据完全归用户所有，安全、可控、极速。

## 6. 开启你的 Orbital 之旅：部署与启动指南

无论你是普通用户还是开发者，启动 Orbital Notes 都非常简单。

### 📦 方式 A：直接使用（推荐非开发者）
1.  前往项目的 Release 页面。
2.  下载最新版本的 `Orbital Notes 1.0.0.exe`。
3.  双击运行，小机器人即可在你的桌面起航。

### 🛠️ 方式 B：开发者模式（进阶定制）
如果你想根据源代码进行二次开发或构建：
1.  **克隆项目**：
    ```bash
    git clone https://github.com/your-username/orbital-notes.git
    cd orbital-notes
    ```
2.  **安装依赖**（确保已安装 Node.js）：
    ```bash
    npm install
    ```
3.  **启动开发版**：
    ```bash
    npm start
    ```
4.  **打包成 exe**：
    ```bash
    npm run build
    ```

## 7. 详细使用教程：从入门到效率大神

### 7.1 悬浮机器人与 Todo List
*   **隐藏/唤回**：按 `Alt + 1` 让它随叫随到。
*   **拖拽**：鼠标左键按住机器人即可满屏幕飞。
*   **待办管理**：鼠标悬停 1 秒，小机器人旋转并微笑，滑出待办窗口。
    *   点击 **+** 快速增加任务。
    *   勾选完成任务。
    *   点击 **“导出为 Note”**，系统会自动将已完成的任务打包成一篇带日期的笔记，并打上 `#Todo List` 标签，方便日后回顾。

### 7.2 快速笔记的“骚操作”
*   **左键单击**：直接打开 Quick Note 窗口。
*   **重命名标题**：觉得 "Quick Note" 字样太单调？**双击标题**即可直接重命名。
*   **自动保存**：无需担心丢失，关闭即存。

### 7.3 强大的笔记管理器
*   **分栏调整**：左右面板比例不合适？鼠标悬停在分界线上，**自由拖拽**。
*   **批量模式**：点击列表下方的“列表复选框”图标，进入批量模式。你可以一次性勾选 10 篇笔记进行**批量删除**或**批量导出（一次性打包所有 MD/PDF）**。

### 7.4 Markdown 高级玩家
除了标准语法，你还可以使用以下自定义组件：

**提示框组件**：
```markdown
<Note>
💡 这里是一条重要提示，它会在预览中以高亮黄色背景展示。
</Note>
```

**折叠面板组件**：
```markdown
<Accordion title="点击查看代码详情">
这里可以放置很长但平时不需要展示的内容。
</Accordion>
```

### 7.5 AI 每日新闻配置
1.  在设置中填入你的 **Dify Base URL** 和 **API Key**。
2.  点击主界面左下角的 **“AI Daily News”**。
3.  Orbital 将自动触发工作流，将最新的 AI 趋势以笔记形式呈现在你面前。
